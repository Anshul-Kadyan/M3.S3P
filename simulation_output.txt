3. COMPILATION REQUIREMENTS
---------------------------
To run these programs, you need:
- OpenCL SDK installed
- C++ compiler (g++, Visual Studio, or MinGW)
- OpenCL-compatible device

Installation Commands:
- Install MinGW-w64 or Visual Studio
- Install Intel OpenCL SDK or vendor-specific OpenCL drivers
- Add compiler to system PATH

Compilation Commands:
g++ -std=c++11 vector_ops.cpp -lOpenCL -o vector_ops.exe
g++ -std=c++11 vector_add.cpp -lOpenCL -o vector_add.exe
